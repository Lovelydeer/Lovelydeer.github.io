每种灯泡必须用不同的电源，

同种灯泡可以共用一个电源，电压低的可以换成电压高的，但反之不然。

1307C. Cow and Message

1307C. Cow and Message

题意：给定一个字符串s，让你找一个子序列出现最多次，而且子序列的下标为等差数列。

嗯，我当时是懵了，还读错了题。

等差序列长度为1或2时很easy；

长度为1时，等差任意，长度为2时，等差也可任意。

当长度为3时事情就变得复杂，因为每一个长度为3的都要对应一个长度为2的序列，而且还要等差。

所以只用判断长度为1 或 2 的子序列即可。

先用 前缀和 算一下之前出现的各个字母数量。

然后在把每一个位置作为后者（即子序列ij中的 j ）

代码和讲的并不对应，只是思考思路而已。

发现口胡错了会update

/********************************************************************************************************************************************************/

Tutorial:

We observe that if the hidden string that occurs the most times has length longer than 22,
then there must exist one that occurs just as many times of length exactly 22.
This is true because we can always just take the first 22 letters;
there can’t be any collisions.
Therefore, we only need to check strings of lengths 11 and 22.
Checking strings of length 11 is easy.
To check strings of length 22,
we can iterate across S from left to right
and update the number of times we have seen each string of length 11 and 22 using DP.

Time Complexity: O(|s|c)O(|s|c) (c is length of alphabet)

/*******************************************************************************************************************************************************/
